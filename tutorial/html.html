Introduction To Web Development

When making websites, you will use to main languages: HTML, CSS, JS. HTML stands for Hyper Text Markup Language, and it will make up the frame of your site. HTML is made of tags. A simple hello world program looks like this:

<!DOCTYPE html>
<html>

	<head>
		<meta  charset="utf-8">
		<meta  name="viewport"  content="width=device-width">
		<title> Hello World! </title>
	</head>
	
	<body>
		<h1> Hello World! </h1>
	</body>
</html>

Next is CSS. This one stands for Cascading Style Sheets. It makes your site look good. This is how you change the font size.

h1 {
	font-size: 15px;
}

Finally is JS, which is JavaScript. This makes your page interactive.

alert("Hello World!")

Section 2 (1.1.2)

How did that code work?
Well first you have the <DOCTYPE HTML>, that tells the browser you are using HTML 5, and not another version like HTML 4, which you would use this: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">, it’s much more complicated because it must refer to a Document Type Definition.

Note that the <DOCTYPE HTML> is NOT an HTML tag.

The next one you see is <html>. Look closely and you will see that there is a </html> too. That is called a tag. Most tags have a opening tag, the first one, and a closing tag, the one with the /. The html tag is the container for all html elements (An element is a basically an pair of html tags) except for the <DOCTYPE HTML>.

Then you have the head element, which contains two meta tags and a title tag. Things inside the <head> tag are usually elements for the browser, such as the encoding of the page, the title of the page, etc…
If you put a element that the browser will display, such as a header ( We’ll get to headers soon.) the browser will still display it, but it is bad practice to do that.

Inside of the head tag, you have the <meta> tag. The meta tag is one of the exceptions to the rule that elements must have a closing tag. You might notice that the inside the opening tag, you have more then just the word meta. You have these things called attributes. These are declared inside the opening tag, like this:

<tag-name attribute-name="attribute-value" ></tag-name>

Attributes are basically like properties. In the example, we have these two meta tags:

<meta charset="utf-8">  
<meta name="viewport" content="width=device-width">

As we can see, we have the charset attribute, the name attribute, and the content attribute. The charset attribute tells the browser what character encoding the page is in.
The name attribute is an attribute used with content that tells what kind of value the content attribute holds.
The content attribute is an attribute used with the name attribute above the holds the content for the name attribute.

You can see more details in the HTML reference section.
