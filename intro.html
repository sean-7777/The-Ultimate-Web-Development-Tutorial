Introduction To Web Development

When making websites, you will use to main languages: HTML, CSS, JS. HTML stands for Hyper Text Markup Language, and it will make up the frame of your site. HTML is made of tags. A simple hello world program looks like this:

<!DOCTYPE html>
<html>

	<head>
		<meta  charset="utf-8">
		<meta  name="viewport"  content="width=device-width">
		<title> Hello World! </title>
	</head>
	
	<body>
		<h1> Hello World! </h1>
	</body>
</html>

Next is CSS. This one stands for Cascading Style Sheets. It makes your site look good. This is how you change the font size.

h1 {
	font-size: 15px;
}

Finally is JS, which is JavaScript. This makes your page interactive.

alert("Hello World!")
